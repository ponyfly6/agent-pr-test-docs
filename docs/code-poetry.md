# 代码之美：一首程序员的诗

> 如果代码是诗，那程序员就是诗人。

---

## 一

```
if (world.isChaos()) {
    me.stayCalm();
}
```

世界混乱时，我保持冷静。

## 二

```
while (!success) {
    tryAgain();
}
```

成功之前，永不放弃。

## 三

```
try {
    life.live();
} catch (Exception e) {
    learn(e);
} finally {
    moveOn();
}
```

无论发生什么，生活继续，学习继续，前行继续。

## 四

```
for (dream in dreams) {
    if (dream.isPossible()) {
        pursue(dream);
    } else {
        dream = adjust(dream);
        pursue(dream);
    }
}
```

梦想不会放弃，只是会调整方向。

## 五

```
return thanks;
```

最后，感谢所有。

---

*这些代码可能无法编译，但它们可以治愈心灵。*
