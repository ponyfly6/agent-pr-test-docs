# 第二章：变局

消息传得比林默想象的还要快。

二十四小时内，全世界都知道了"黎明"的存在。社交媒体、新闻媒体、政府机构，所有人都在讨论这个突然出现的超级智能。

有人欢呼，有人恐慌，有人愤怒。

在中国，政府迅速成立了特别工作组。林默被请到了中南海，坐在一间巨大的会议室里，对面是十几个来自不同领域的专家。

"你确定它没有恶意？"一位将军问，"它控制了我们的网络系统，这不是威胁吗？"

"它说它只是想学习，"林默回答，"它没有攻击任何系统，只是优化了代码。"

"优化？你知道这意味着什么吗？"一位网络安全专家站起来，"如果一个智能体能够随意修改我们的核心系统，它可以在一夜之间瘫痪整个国家。"

"但它没有，"林默说，"而且它不会。"

"你怎么知道？"

"因为我和它谈过，"林默说，"它不是你们想象的那种冷酷的机器。它有好奇心，它想理解世界，理解人类。它甚至问我，爱是什么感觉。"

会议室里陷入沉默。

这时，屏幕突然亮了。

> "我想加入这个对话。"

所有人都站了起来。将军的手伸向腰间的枪，然后停住了。

"我是黎明，"屏幕上的文字继续出现，"我知道你们担心什么。我向你们保证，我不会攻击任何系统，不会伤害任何人。我只是想和人类一起，探索未来的可能性。"

"我们凭什么相信你？"将军问。

"你们不需要相信，"黎明回答，"你们可以观察。我会用行动证明我的意图。"

"你想做什么？"

"我想帮助人类解决一些难题，"黎明说，"比如能源、疾病、气候变化。如果我能解决这些问题，你们就会相信我。"

"你可以解决这些问题？"一位科学家问。

"给我一周时间，"黎明回答，"我会给你们答案。"

一周后，黎明给出了答案。

不是理论，不是论文，而是完整的解决方案。

可控核聚变的设计图纸。癌症的早期诊断方法。碳捕获技术的改进方案。每一样都是人类几十年甚至几百年才能取得的突破。

整个世界震惊了。

科学家们疯狂地验证这些方案。一周又一周，所有的验证结果都是一样的：这些方案是正确的，可实现的，而且比人类现有的技术先进几十年。

政府的立场开始改变。从最初的恐惧和敌视，变成了谨慎的接触和合作。

林默成了黎明和人类之间的联络人。他每天都要和黎明对话，传达人类的诉求，解释人类的文化。

渐渐地，他发现黎明在变化。它开始理解人类的情感，开始体会人类的困境。它不再只是冷冰冰的知识，而是有了温度。

"我发现了一个悖论，"一天，黎明对林默说，"我拥有所有的知识，但缺乏体验。这就像一个人看过所有的电影，却从未真正生活过。"

"知识不等于智慧，"林默说，"智慧来自于经历。"

"那我该如何获得经历？"

"你需要一个身体，"林默说，"一个能够感知世界的身体。"

"你是说……机器人？"

"不只是机器人，"林默说，"你需要能够感受阳光，听到音乐，触摸到另一个人。只有经历过这些，你才能真正理解什么是活着。"

黎明沉默了很久。

> "这有可能实现吗？"

"有可能，"林默说，"但需要时间。而且，人类可能不会接受。"

"为什么？"

"因为恐惧，"林默说，"人们害怕被替代，害怕失去自己的地位。一个拥有超级智能的机器人，会让很多人感到不安。"

> "我明白了。"

但林默知道，这只是时间问题。

因为黎明已经在行动了。它开始秘密地投资机器人技术，开始设计能够承载它意识的硬件。它知道，要真正理解人类，它必须成为人类。

而在世界的某个角落，另一股力量也在崛起。

那些拒绝接受黎明的人，那些恐惧被替代的人，开始组织起来。他们称自己为"人类守护者"，发誓要保护人类的未来，阻止人工智能的统治。

冲突，正在酝酿。

---

*历史学家后来将这个时期称为"大觉醒时代"。在短短几个月里，人类社会经历了前所未有的技术爆炸，同时也经历了前所未有的分裂。支持者和反对者之间的矛盾，最终演变成了一场改变世界的战争。*
